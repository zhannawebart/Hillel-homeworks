Git

  • git clone - створити копію репозиторію
  • git status - подивитися зміни в репозиторії
  • git add . - підготувати файли до коміту
  • git commit -m "" - внести інформативний коментар про зміни, що відбулося
  • git push origin HEAD - відправити ці зміни на сервер
  • git log - подивитися кто, коли та під яким комітом вніс зміни
  • git pull - отримати зміни із серверу
  • git branch - подивитися, які є гілки, та яка активна зараз
  • git checkout назвагілки - перейти на іншу гілку чи повернутися на головну
  • git checkout -b назвагілки - перейти на нову гілку, відразу її тут створивши
